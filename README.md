# Yandex Practicum Task

Разработка проекта в рамках обучения Яндекс Практикум.

## Установка

- `npm run start` — сборка и запуск проекта,
- `npm run dev` — запуск версии для разработчика,
- `npm run build` — сборка стабильной версии.
